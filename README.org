#+TITLE: AUTOMATED MUNKI CLIENT CONFIGURATION
#+AUTHOR: Toby Slight

*This script must be run as root. It carries out the following tasks:*

  1) Create new Munki manifest with machine's serial number as name.
  2) Choose catalogs/included manifests to add newly created manifest to.
  3) Check for and display available software (-i only)
  4) Install all available software.
